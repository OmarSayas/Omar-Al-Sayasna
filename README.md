## 🚀 About Me

I build software that works in the real world — from C#/.NET backend systems and Angular frontends to computer vision models and RFID/smartcard configuration tools. Right now I'm finishing my CS degree while working at Inepro ID, where I'm developing a desktop application that automatically configures smartcard readers using pattern detection across multiple card technologies including MIFARE, DESFire, and Legic. Before that I spent time at an exchange semester in South Korea at Yonsei University, which gave me a sharper eye for working across different team cultures and communication styles.

I pick up new tools fast, prefer clean architecture over clever hacks, and like working on problems that have actual users at the end of them.

## 🧠 Featured Projects

### 🔹 Festivia — Festival Management Platform
A full-stack platform for managing festivals and events. Users can browse events, purchase tickets, and receive confirmation emails with QR codes for entry. The system includes a complete admin panel and a CRM with a WYSIWYG editor for managing content and communication.

👉 Repo: https://github.com/OmarSayas/Festivia  
👉 Tech: PHP, Vue, MySQL, Docker

---

### 🔹 Mycelium Growth AI Model
A computer vision system that predicts the growth stage of mycelium based on image data. The model combines deep learning and clustering techniques to identify growth patterns over time, achieving **96% accuracy** across exact and near-correct predictions.

👉 Repo: https://github.com/OmarSayas/Mycelium-Growth  
👉 Tech: Python, TensorFlow, VGG16, HDBSCAN, PCA

---

### 🔹 Full-Stack Webshop
